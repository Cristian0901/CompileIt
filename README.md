# Compile It

It is small script which can compile and excute your programs quickly.
---
## Supported languages
* C
* C++
* Go
* Rust
* Java
---
## Requirements
You just need to have the compilers that you need.
Supported compilers:

- gcc (C files)
- g++ (C++files)
- Go (Go files)
- Rust (Rust files)
